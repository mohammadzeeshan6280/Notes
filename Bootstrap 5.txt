Bootstrap 5 : Overview		Chapter 1
What is Bootstrap=>
Quickly design and customize responsive mobile-first sites with Bootstrap. the
world's, most popular front-end open source toolkit. Sass variables and mixins
responsive grid system, extensive pre-built components, and powerful JavaScript
plugins.

In simple terms=>
(i). Bootstrap is a giant collection of handy, resuable bits of code written in HTML,CSS, JavaScript.
(ii). it's also a front-end development framework that enables developers & designers to quickly
build fully responsive websites
(iii). Bootstrap is completely free to use in personal and commercial projects.
(iv). Bootstrap saves you from writing lots of CSS code.

Bootstrap Layouts and Grids System	Chapter 4
Basic Terminology
1. What is Responsive Design ?
i) Responsive Web design is the approach that suggests that design and development should respond to
   the user,s behaviour and environment based on screen size. platform and orientation
ii) The practice consists of a mix of flexible grids and layouts, images and an inteligent use of CSS media queries.

2. What is a Box Model ?
i) In CSS, the term "Box Model" is used when talking about design and layout. The CSS box model is essentially 
   box that wraps around every HTML element. it consists of: Margins, Borders, Padding, and the actual content

3. Bootstap Container Layout
i) Container are the most basic layout element in Bootstrap.
ii) Containers are used to pad the content inside of them.
iii) There are two container classes available:
iv) The .Container class provides a responsive fixed width container
v) The .Container-fluid class provides a full width container, spanning the entire
    width of the viewport

4. Container-Fluid
i) Width is always 100%
